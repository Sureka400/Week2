Project Overview This project uses a Convolutional Neural Network (CNN) to classify waste images into categories like plastic, paper, metal, and organic.
Built using TensorFlow and deployed using Streamlit Cloud.

Try the app here ["https://zj9fxtmslwykbhxgehhppx.streamlit.app/"]

Technologies Used

Python 3.x
TensorFlow / Keras
Streamlit
NumPy, Pillow, OpenCV
Google Colab for model training
